## 미완성 항목 / 주의사항

1. **ANTHROPIC_API_KEY**: 환경변수가 User 레벨에만 등록됨. Task Scheduler에서 실행 시 환경변수를 못 읽을 수 있음 → `run_daily.bat`에서 직접 설정하거나, 시스템 레벨로 등록 필요
2. **watcher.py 상시 실행**: 현재 foreground 실행만 가능. 상시 실행하려면 Windows 서비스로 등록하거나, 시작 시 자동 실행 설정 필요
3. **PDF 지원**: pymupdf(fitz) 미설치 시 PDF 파일 처리 불가. 필요 시 `pip install pymupdf`

## 내일 사용법

### 수동 퀴즈 생성
```bash
cd "_pipeline/scripts"
python generate.py "../../3학년 1학기/중간고사/유기화학/필기/4월 9일.md"
```

### 폴더 전체 처리
```bash
python generate.py "../../3학년 1학기/중간고사/유기화학/"
```

### 실시간 감시 모드
```bash
python watcher.py
# 새 필기 파일 저장 시 자동으로 퀴즈 생성
# Ctrl+C로 종료
```

### 퀴즈 리뷰
```bash
python review.py list     # queue 목록
python review.py do       # 대화형 풀기
python review.py report   # 취약 개념 리포트
```

### 스케줄러 관리
```bash
python scheduler.py status     # 등록 상태 확인
python scheduler.py run-now    # 즉시 실행 (오늘 수정 파일만)
python scheduler.py unregister # 등록 해제
python scheduler.py register   # 재등록
```
