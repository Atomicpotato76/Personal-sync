# Study Pipeline 사용법 가이드

> AI가 내 필기를 읽고, 요약 노트·퀴즈·복습 일정을 자동으로 만들어주는 개인 학습 시스템입니다.

---

## 이런 흐름으로 돌아갑니다

```
내가 Obsidian에 필기 저장
        ↓
AI가 자동으로 처리 (교재 참고 + 논문 검색 + 요약 + 퀴즈 생성)
        ↓
퀴즈를 풀고 결과를 기록
        ↓
AI가 내 취약점을 파악해서 복습 일정을 짜줌
```

---

## 기능 1 — 자동 노트 처리 (백그라운드 자동 실행)

**언제**: Obsidian 필기 폴더(유기화학·유전체학과AI·응용균학)에 `.md` 파일을 저장하는 순간

**무엇을 해주나**:
- 교재 해당 챕터를 자동으로 찾아서 함께 참고
- 관련 논문(PubMed·Semantic Scholar)을 검색해서 내용 보강
- 내 필기에서 빠진 개념을 파악하여 보충 설명 추가
- 정리된 요약 노트를 마크다운·PDF 두 가지로 저장 (output/)
- 퀴즈를 자동 생성해서 대기 목록(queue/)에 추가

> 별도로 할 게 없습니다. 저장하면 끝입니다.

---

## 기능 2 — 퀴즈 풀기 (CLI)

터미널에서 실행:

```
python review.py do
```

- 대기 중인 퀴즈가 하나씩 출제됩니다
- 답을 생각한 뒤 Enter, 모범 답안 확인 후 `c(맞춤) / w(틀림) / p(부분정답) / s(건너뜀)` 입력
- 틀렸을 때 짧은 메모를 남길 수 있습니다
- 완료되면 approved/ 폴더로 이동되고 복습 일정이 자동으로 갱신됩니다

**대기 목록 확인만 하고 싶을 때**:

```
python review.py list
```

---

## 기능 3 — 취약 개념 리포트 (CLI)

```
python review.py report
```

- 과목별로 개념마다 마스터리 수치(0~100%)를 바 그래프로 표시
- 우선순위(높음 🔴 / 중간 🟡 / 낮음 🟢)와 최근 오답 메모를 함께 보여줍니다

---

## 기능 4 — 복습 일정 (Hermes)

**오늘 할 일 확인**:

```
python hermes.py plan-day
```

**이번 주 계획 보기**:

```
python hermes.py plan-week
```

- 틀렸던 개념은 더 자주, 잘 아는 개념은 간격을 늘려서 배치합니다 (간격반복)
- 시험·과제 마감이 가까울수록 해당 과목 비중이 높아집니다

**시험 일정 추가**:

```
python hermes.py add-exam organic_chem 2026-05-10 "중간고사"
```

## 기능 5 — 웹 대시보드

```
streamlit run scripts/dashboard/app.py
```

브라우저(localhost:8501)에서:

| 탭 | 하는 일 |
|----|---------|
| 퀴즈 관리 | 대기 중인 퀴즈를 보고 승인·거절 |
| 파이프라인 실행 | 특정 노트를 수동으로 처리 |
| 설정 편집 | LLM 연결·과목·경로 설정 변경 |
| 리포트 | 학습 세션 요약 생성 |

## 기능 6 — Claude Desktop 연동 (MCP)

Claude Desktop 앱 채팅창에서 말로 아래 작업을 할 수 있습니다:

| 무엇을 할 수 있나 | 예시 표현 |
|-----------------|-----------|
| 노트 검색 | "SN2 반응에 대한 내 필기 찾아줘" |
| 개념 설명 요청 | "카보양이온 안정성 설명해줘" |
| 즉석 퀴즈 | "유기화학 퀴즈 내줘" |
| 취약 개념 확인 | "내가 요즘 뭘 제일 못해?" |
| 오늘 일정 확인 | "오늘 뭐 공부해야 해?" |
| 시험 일정 추가 | "5월 10일 유기화학 중간고사 추가해줘" |
| 결과 기록 | "방금 SN2 틀렸어" |
| 교재 조회 | "유기화학 3챕터 내용 보여줘" |

> Claude가 내 취약도(mastery)를 보고 설명 깊이를 자동으로 조절합니다.
> 잘 모르는 개념이면 기초부터, 어느 정도 알면 핵심만, 숙달된 개념은 심화·시험 포인트 위주로 설명합니다.

## 복습 일정 계산 방식

퀴즈 결과에 따라 다음 복습 날짜가 자동으로 바뀝니다:

| 결과 | 다음 복습 |
|------|-----------|
| 맞춤 (처음) | 6일 후 |
| 맞춤 (반복) | 점점 더 긴 간격 |
| 부분 정답 | 현재 간격의 70% |
| 틀림 | 다음 날 바로 |

## 과목별 폴더 구조

| Obsidian 폴더 | 과목 |
|--------------|------|
| 유기화학/ | 유기화학 |
| 유전체학과AI/ | 유전체학과 AI |

## 파일 위치 요약

| 폴더/파일 | 내용 |
|-----------|------|
| `output/md/` | 생성된 요약 노트 (마크다운) |
| `output/pdf/` | 생성된 요약 노트 (PDF) |
| `queue/` | 풀기 전 퀴즈 대기 목록 |
| `approved/` | 완료된 퀴즈 보관 |
| `weak_concepts.json` | 개념별 마스터리·복습 일정 |
| `cache/learning_history.json` | 전체 학습 이력 |
